🔗 API ENDPOINTS (15+)

Authentication:
  POST /auth/register → Create account
  POST /auth/login → Login

SOS Emergency:
  POST /sos/trigger → Send emergency
  GET /sos/active → Active incidents
  GET /sos/user/{id} → User incidents
  POST /sos/resolve/{id} → Resolve incident

Community:
  POST /community/report → Create report
  GET /community/reports/risk/{level} → By risk level
  GET /community/reports/verified → Verified reports
  GET /community/reports/nearby → Nearby reports
  POST /community/reports/{id}/upvote → Upvote report

Contacts:
  POST /contacts/add → Add contact
  GET /contacts/user/{id} → Get contacts
  DELETE /contacts/{id} → Delete contact

Zones:
  GET /zones/all → All zones
  GET /zones/type/{type} → By type
  GET /zones/nearby → Nearby zones

🚀 QUICK START (5 MINUTES)

1. Setup Database (1 min)
   mysql -u root -p
   CREATE DATABASE safecircle;
   source database.sql;

2. Configure Backend (1 min)
   Edit: Backend/src/main/resources/application.properties
   Set MySQL password

3. Start Backend (1 min)
   cd Backend
   mvn spring-boot:run

4. Open Frontend (1 min)
   Open: Frontend/index.html
   Or: python -m http.server 8000

5. Test (1 min)
   Create account
   Click SOS button
   Create report
   Done!
