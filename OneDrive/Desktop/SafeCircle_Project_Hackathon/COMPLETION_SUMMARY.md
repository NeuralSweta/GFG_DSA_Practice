## 📞 SUPPORT & MAINTENANCE

### Known Limitations:
- Voice recognition is mock (for demo)
- Shake detection only on physical devices
- Backend API integration required
- Geolocation requires user permission
- HTTPS required for production

### Recommended Setup:
1. Backend: Spring Boot running on port 8080
2. Frontend: Served on localhost:8000 (or any server)
3. Database: MySQL running locally
4. Browser: Chrome/Firefox (latest versions)

### Troubleshooting:
- **SOS not triggering**: Check geolocation permissions
- **Voice SOS not working**: Use Chrome/Edge browsers
- **Shake not detecting**: Ensure physical device motion enabled
- **Contacts not showing**: Check browser storage permissions
