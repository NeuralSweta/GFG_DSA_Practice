# Quick Sort

| Field | Details |
|-------|---------|
| **Topic** | Arrays |
| **Difficulty** | 🟡 Medium |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/quick-sort/0) |
| **Approach** | Divide and Conquer with partition |
| **Time Complexity** | O(n log n) avg |
| **Space Complexity** | O(log n) |

## Key Takeaway
> Pick pivot, partition around it, recurse on both halves.
