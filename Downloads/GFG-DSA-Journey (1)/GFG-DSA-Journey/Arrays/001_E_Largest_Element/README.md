| Field | Details |
|-------|---------|
| **Topic** | Arrays |
| **Difficulty** | 🟢 Easy |

## Problem Summary
Find the largest element in a given array by scanning all elements once.

## Key Takeaway
> Classic linear scan. Initialize `max = arr[0]`, iterate and update.
