# Set Matrix Zeroes

| Field | Details |
|-------|---------|
| **Topic** | Arrays / Matrix |
| **Difficulty** | 🟡 Medium |
| **GFG Link** | [Set Matrix Zeroes](https://www.geeksforgeeks.org/problems/set-matrix-zeroes/1) |
| **Approach** | In-place using first row/col as markers |
| **Time Complexity** | O(m×n) |
| **Space Complexity** | O(1) |

## Key Takeaway
> Use the first row and first column as sentinel arrays. Handle them separately at the end.
