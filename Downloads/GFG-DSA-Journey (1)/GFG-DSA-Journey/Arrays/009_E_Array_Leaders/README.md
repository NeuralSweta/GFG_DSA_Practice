# Array Leaders

| Field | Details |
|-------|---------|
| **Topic** | Arrays |
| **Difficulty** | 🟢 Easy |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/leaders-in-an-array-1587115620/0) |
| **Approach** | Traverse from right, track max |
| **Time Complexity** | O(n) |
| **Space Complexity** | O(1) |

## Key Takeaway
> An element is a leader if it's greater than all elements to its right. Scan right to left.
