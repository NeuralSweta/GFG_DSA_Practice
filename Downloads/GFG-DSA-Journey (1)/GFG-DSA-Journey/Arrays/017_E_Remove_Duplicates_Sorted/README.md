# Remove Duplicates from Sorted Array

| Field | Details |
|-------|---------|
| **Topic** | Arrays |
| **Difficulty** | 🟢 Easy |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/remove-duplicate-elements-from-sorted-array/0) |
| **Approach** | Two pointer in-place |
| **Time Complexity** | O(n) |
| **Space Complexity** | O(1) |

## Key Takeaway
> i=0, j=1. If arr[i]!=arr[j], move arr[i+1]=arr[j].
