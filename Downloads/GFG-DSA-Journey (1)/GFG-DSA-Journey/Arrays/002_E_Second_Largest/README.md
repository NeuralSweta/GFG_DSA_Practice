# Second Largest Element

| Field | Details |
|-------|---------|
| **Topic** | Arrays |
| **Difficulty** | 🟢 Easy |
| **GFG Link** | [Second Largest](https://www.geeksforgeeks.org/problems/second-largest3735/1) |
| **Approach** | Single Pass with two variables |
| **Time Complexity** | O(n) |
| **Space Complexity** | O(1) |

## Problem Summary
Find the second largest distinct element in an array.

## Key Takeaway
> Track `first` and `second` simultaneously. Skip duplicates of `first`.
