# Min and Max in Array

| Field | Details |
|-------|---------|
| **Topic** | Arrays |
| **Difficulty** | ⚪ Basic |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/find-minimum-and-maximum-element-in-an-array4428/0) |
| **Approach** | Single pass tracking min and max |
| **Time Complexity** | O(n) |
| **Space Complexity** | O(1) |

## Key Takeaway
> Initialize min=max=arr[0], scan once updating both.
