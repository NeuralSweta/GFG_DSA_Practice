# Minimum in BST

| Field | Details |
|-------|---------|
| **Topic** | BST |
| **Difficulty** | 🟢 Easy |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/minimum-element-in-bst/0) |
| **Approach** | Go leftmost node |
| **Time Complexity** | O(h) |
| **Space Complexity** | O(1) |

## Key Takeaway
> Keep going left until null. That node has the minimum value.
