# BST to Greater Sum Tree

| Field | Details |
|-------|---------|
| **Topic** | BST |
| **Difficulty** | 🟡 Medium |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/bst-to-greater-sum-tree/0) |
| **Approach** | Reverse inorder traversal (right→root→left) |
| **Time Complexity** | O(n) |
| **Space Complexity** | O(h) |

## Key Takeaway
> Process right subtree first, accumulate sum, update node value.
