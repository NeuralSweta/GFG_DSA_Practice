# Array to BST

| Field | Details |
|-------|---------|
| **Topic** | BST |
| **Difficulty** | 🟡 Medium |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/array-to-bst4443/0) |
| **Approach** | Sort then pick mid as root recursively |
| **Time Complexity** | O(n log n) |
| **Space Complexity** | O(n) |

## Key Takeaway
> Sort array, mid = root, recurse on left and right halves.
