# Largest BST in Binary Tree

| Field | Details |
|-------|---------|
| **Topic** | BST |
| **Difficulty** | 🟡 Medium |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/largest-bst/0) |
| **Approach** | Post-order with size/min/max at each node |
| **Time Complexity** | O(n) |
| **Space Complexity** | O(h) |

## Key Takeaway
> For each subtree return {isBST, size, min, max}. Track max BST size globally.
