# Prime Number

| Field | Details |
|-------|---------|
| **Topic** | Math |
| **Difficulty** | 🟢 Easy |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/prime-number2314/0) |
| **Approach** | Trial division up to √n |
| **Time Complexity** | O(√n) |
| **Space Complexity** | O(1) |

## Key Takeaway
> Loop from 2 to √n. If any divisor found → not prime.
