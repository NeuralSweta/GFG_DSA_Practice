# Number of Factors

| Field | Details |
|-------|---------|
| **Topic** | Math |
| **Difficulty** | 🟢 Easy |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/number-of-factors1435/0) |
| **Approach** | Count divisors up to √n |
| **Time Complexity** | O(√n) |
| **Space Complexity** | O(1) |

## Key Takeaway
> For i up to √n: if i divides n, count 2 factors (i and n/i). If perfect square, count 1.
