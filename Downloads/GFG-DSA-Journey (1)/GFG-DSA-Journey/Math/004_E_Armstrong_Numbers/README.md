# Armstrong Numbers

| Field | Details |
|-------|---------|
| **Topic** | Math |
| **Difficulty** | 🟢 Easy |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/armstrong-numbers2727/0) |
| **Approach** | Sum of digits raised to power of digit count |
| **Time Complexity** | O(d) |
| **Space Complexity** | O(1) |

## Key Takeaway
> Count digits d, then check if sum of (each digit ^ d) == n.
