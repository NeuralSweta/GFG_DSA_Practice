# Count Digits in a Number

| Field | Details |
|-------|---------|
| **Topic** | Math |
| **Difficulty** | 🟢 Easy |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/count-total-digits-in-a-number/0) |
| **Approach** | Divide by 10 repeatedly |
| **Time Complexity** | O(d) |
| **Space Complexity** | O(1) |

## Key Takeaway
> While n > 0: count++, n /= 10. Or use String.valueOf(n).length().
