# Climbing Stairs / Count Ways to Nth Stair

| Field | Details |
|-------|---------|
| **Topic** | Dynamic Programming |
| **Difficulty** | 🟢 Easy |
| **GFG Link** | [Count Ways to Nth Stair](https://www.geeksforgeeks.org/problems/count-ways-to-nth-stair-1587115620/1) |
| **Approach** | Bottom-up DP (Fibonacci pattern) |
| **Time Complexity** | O(n) |
| **Space Complexity** | O(n) → optimizable to O(1) |

## Key Takeaway
> Classic Fibonacci DP. `dp[i] = dp[i-1] + dp[i-2]`
