# Tree Boundary Traversal

| Field | Details |
|-------|---------|
| **Topic** | Trees |
| **Difficulty** | 🟡 Medium |
| **GFG Link** | [Solve](https://www.geeksforgeeks.org/problems/boundary-traversal-of-binary-tree/0) |
| **Approach** | Left boundary + Leaves + Right boundary |
| **Time Complexity** | O(n) |
| **Space Complexity** | O(h) |

## Key Takeaway
> 3 passes: left boundary (top-down), all leaves (L→R), right boundary (bottom-up).
