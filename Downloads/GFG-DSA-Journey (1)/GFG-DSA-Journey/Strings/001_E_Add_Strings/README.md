# Add Strings

| Field | Details |
|-------|---------|
| **Topic** | Strings |
| **Difficulty** | 🟢 Easy |
| **GFG Link** | [Add Strings](https://www.geeksforgeeks.org/problems/add-strings/1) |
| **Approach** | Simulate digit-by-digit addition with carry |
| **Time Complexity** | O(max(m, n)) |
| **Space Complexity** | O(max(m, n)) |

## Key Takeaway
> Traverse from right to left, track carry. Use `char - '0'` to convert digit char to int.
